Abstract

DinarScript is a statically analyzed programming language inspired by the Kuwaiti Dinar. It combines custom currency types, pattern matching, records, and expressive control flow through a uniquely Kuwaiti lens.

Developer

DinarScript was created from a passion for combining creativity, finance, and programming into one original language concept. Inspired by the idea of giving code a unique identity tied to Kuwaiti currency, the project was designed to showcase how programming languages can carry both technical structure and personal vision.

Swift is the language I enjoy working with the most, which strongly influenced my appreciation for clean syntax, modern design, and elegant user experiences. That mindset helped shape both the language concept and the interactive website experience behind DinarScript.

To explore the source code, development progress, compiler structure, and technical details behind DinarScript, click the GitHub link on the top right of the screen.
